# Job Portal Django Application

This project is a Job Portal application built with Django, PostgreSQL, and Postman for API testing.

## Requirements

- Python 3.8+
- Django 3.x+
- PostgreSQL
- pip (Python package installer)
- Postman (for testing API endpoints)
